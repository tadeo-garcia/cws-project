# cws-project
cws project 

## Features:

#### 1. Choose Time / Meeting Type

#### 2. Host Event
    - Options:
        - Time
        - Type
        - Theme

#### 3. Join Event 
    - Reminder for upcoming events
    - Suggest items to go along with event
    - Suggest other events you might like based on history

#### 4. Dashboard - Joined/Events
    - Google API

#### 5. Edit Account
